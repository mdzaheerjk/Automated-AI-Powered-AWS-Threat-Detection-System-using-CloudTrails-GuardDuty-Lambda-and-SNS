# Automated-AI-Powered-AWS-Threat-Detection-System-using-CloudTrails-GuardDuty-Lambda-and-SNS
This project creates an automated AI-driven threat detection and response pipeline on AWS. Amazon GuardDuty detects suspicious activities, EventBridge routes alerts to Lambda for analysis and remediation steps, and Amazon SNS sends real-time security notifications to administrators via email and SMS, enabling faster incident response
